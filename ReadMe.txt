Список файлов проекта:

1. sanbox.R - Черновик для настройки модели, подбора признаков.

2. final.R - Окончательный вариант. Выполняет обучение модели и создает отправленное на чемпионат решение. В целом повторяет sanbox.R, но содержит только окончательный набор признаков, удалены следы выбора модели.
